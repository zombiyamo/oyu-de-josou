# お湯de除草 〜砂利の庭〜 🫖🌿

砂利の隙間から生えてくる雑草に熱湯をかけて枯らす、癒し系の箱庭ブラウザゲームです。
葉っぱにお湯がかかると少し色が変わり、しばらく眺めていると だんだん黄色く、茶色く枯れていきます。

## 遊び方

`index.html` をブラウザで開くだけで遊べます(外部依存なし・1ファイル完結)。

- **長押し**(マウスのボタンを押しっぱなし / 画面を指で押しっぱなし)で、やかんからお湯が注がれます
- お湯はかけ放題。心ゆくまでどうぞ
- お湯がかかった葉はすぐに色がくすみ、その後ゲーム内の数日(現実の数十秒)かけて枯れていきます
- 枯れ切った草は「枯れ草」としてその場に残ります。タップ(お湯をかける)とホロッと崩れて消えます
- 時間が経つと、また新しい草が生えてきます(画面上部に日数カウンター)
- 草は同じ種類がまとまって群生し、マット状に茂ります(草自体は成長せず、生えた時から完成サイズです)

## 雑草の種類

| 見た目 | 名前 | 特徴 |
|---|---|---|
| 細い葉がツンツン | 若草(イネ科) | さっとかけるだけですぐ枯れる |
| 細い茎が這って小さな葉がつながる | メヒシバ系 | 匍匐茎の先で根付いてどんどん連なって広がる |
| クローバー風の三つ葉・黄色い花 | カタバミ | わりとすぐ枯れる |
| ギザギザの葉のロゼット | タンポポ系 | 少し多めにかける必要がある |
| 丸く広い葉・平行脈 | オオバコ | 踏まれ強いだけあってやや頑丈 |
| 針のような細い葉が放射状 | スギナ | 見た目より頑丈 |
| ハート形の葉・白い花 | ドクダミ系 | **しっかり長めに**かけないと根が生き残り、数十秒後に同じ場所から復活する。根まで火が通ると湯気がブワッと立ちのぼる |

ちょっとだけお湯がかかった草は、そのまま持ち直すこともあります。本物と同じです。
