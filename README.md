# お湯de除草 〜砂利の庭〜 🫖🌿

**▶ 遊ぶ: https://zombiyamo.github.io/oyu-de-josou/**

砂利の隙間から生えてくる雑草に熱湯をかけて枯らす、癒し系の箱庭ブラウザゲームです。
葉っぱにお湯がかかると少し色が変わり、しばらく眺めていると だんだん黄色く、茶色く枯れていきます。

## 遊び方

- **長押し**(マウスのボタンを押しっぱなし / 画面を指で押しっぱなし)で、やかんからお湯が注がれます
- お湯はかけ放題。心ゆくまでどうぞ
- お湯がかかった葉はすぐに色がくすみ、その後ゲーム内の数日(現実の数十秒)かけて枯れていきます
- 枯れ切った草は「枯れ草」としてその場に残ります。タップするとホロッと崩れて消えます
- 時間が経つと、また新しい草が生えてきます(画面上部に日数カウンター)
- 草は同じ種類がまとまって群生し、マット状に茂ります
- **ゴールは全13種の雑草コレクション完成**。初回起動時にかんたんな案内が表示されます

ちょっとだけお湯がかかった草は、そのまま持ち直すこともあります。本物と同じです。

## 雑草コレクション

この庭には、ぜんぶで**13種類**の雑草が生えるらしい——

- 最初に出会えるのは3種類だけ。**除草した数が増えると、新しい種類が生えてくるようになります**
- 解放した雑草は、右上の📖ボタンの**コレクション**で図鑑カード(絵・枯らした数・特徴)として読めます
- 最後のひとつは…?
- あそびかたは右上の?ボタンでいつでも確認できます

## ボーナス

枯れ草を片付けていると、たまに地面にアイテムが落ちます。**タップで拾うと即発動**:

- ✨ **金のやかん**(たまに落ちる): 20秒間やかんが2つになり、注ぎ位置の反対側にも同時にお湯が注がれる
- 🧪 **除草剤**(めったに落ちない): 発動地点から波が広がり、届いた草が根こそぎ枯れて崩れる。**コモン(緑)・アンコモン(青)・レア(紫)**のレア度があり、レアほど枯れる割合が高く、めったに落ちない

## ホームセンター 🛒

草を1本片付けるごとに**除草ポイント**が1ptたまります(使っても図鑑の進行は減りません)。
右上の🛒ボタンのホームセンターで、ポイントと引き換えに**自動化アイテム**が買えます:

- 🫖 **自動やかん**: 庭に置くと、近くの草にときどきお湯をかけてくれる
- 💦 **熱湯スプリンクラー**: まわりの草にまとめて熱湯をまく
- 🤖 **ヤギ型除草ロボ**: 庭をのんびり歩き回って、枯れ草をもぐもぐ回収してくれる(生身のヤギは熱湯が危ないのでロボットです)

買うたびに値段が上がります。庭が勝手に片付いていく様子を、お茶でも飲みながら眺めましょう。
