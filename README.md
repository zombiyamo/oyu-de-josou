# お湯de除草 〜砂利の庭〜 🫖🌿

**▶ 遊ぶ: https://zombiyamo.github.io/oyu-de-josou/** (GitHub Pagesの有効化が必要。手順は下記)

砂利の隙間から生えてくる雑草に熱湯をかけて枯らす、癒し系の箱庭ブラウザゲームです。
葉っぱにお湯がかかると少し色が変わり、しばらく眺めていると だんだん黄色く、茶色く枯れていきます。

## 遊び方

`index.html` をブラウザで開くだけで遊べます(外部依存なし・1ファイル完結)。

- **長押し**(マウスのボタンを押しっぱなし / 画面を指で押しっぱなし)で、やかんからお湯が注がれます
- お湯はかけ放題。心ゆくまでどうぞ
- お湯がかかった葉はすぐに色がくすみ、その後ゲーム内の数日(現実の数十秒)かけて枯れていきます
- 枯れ切った草は「枯れ草」としてその場に残ります。タップ(お湯をかける)とホロッと崩れて消えます
- 時間が経つと、また新しい草が生えてきます(画面上部に日数カウンター)
- 草は同じ種類がまとまって群生し、マット状に茂ります(草自体は成長せず、生えた時から完成サイズです)
- **ゴールは全13種の雑草コレクション完成**。初回起動時にかんたんな案内が表示されます

## 雑草コレクション

この庭には、ぜんぶで**13種類**の雑草が生えるらしい——

- 最初に出会えるのは3種類だけ。**除草した数が増えると、新しい種類が生えてくるようになります**
- 解放した雑草は、右上の📖ボタンの**コレクション**で図鑑カード(絵・枯らした数・特徴)として読めます
- 最後のひとつは…?
- あそびかたは右上の?ボタンでいつでも確認できます

ちょっとだけお湯がかかった草は、そのまま持ち直すこともあります。本物と同じです。

## ボーナス

枯れ草を片付けていると、たまに地面にアイテムが落ちます。**タップで拾うと即発動**:

- ✨ **金のやかん**(たまに落ちる): 20秒間やかんが2つになり、注ぎ位置の反対側にも同時にお湯が注がれる
- 🧪 **除草剤**(めったに落ちない): 発動地点から波が広がり、画面じゅうの草が根こそぎ枯れて崩れる。もちろん全部カウントされる

## 公開手順(GitHub Pages)

1. このリポジトリの **Settings → Pages** を開く
2. 「Build and deployment」の Source を **Deploy from a branch** にする
3. Branch: `claude/weed-removal-browser-game-lyop0n` / フォルダ `/ (root)` を選んで **Save**
4. 数分待つと `https://zombiyamo.github.io/oyu-de-josou/` で遊べるようになります

補足:
- OGPタグ設定済み。X(Twitter)にURLを貼ると大きなサムネイルカード(`ogp.png`)が表示されます
- PWA対応済み。スマホで開いて「ホーム画面に追加」すると全画面のアプリとして遊べ、一度読み込めばオフラインでも動きます
- リポジトリ名を変えたい場合(URLを変えたい場合)は、`index.html` 内の `og:url` / `og:image` のURLも合わせて変更してください
